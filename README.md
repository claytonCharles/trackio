# Trackio 📦

Trackio será um sistema de monitoramento interno para controle de equipamentos em geral de TI, sendo possível auditar, 
validar, documentar e afins...

A ideia está em desenvolvimento tanto no conceito quanto no código em si, sendo nessa fase inicial totalmente volátil e 
podendo mudar a qualquer instante. Ao chegar a uma base estável, este documento será atualizado, juntamente com um 
versionamento estável da mesma.
